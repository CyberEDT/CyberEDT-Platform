# CyberEDT Website

A modern, responsive, and cybersecurity-first platform for CyberEDT — a startup focused on **Education, Defence, and Tools (E-D-T)**.

This website serves as the core digital identity of CyberEDT, showcasing its ecosystem, tools, and services with a clean and premium user experience.

---

## ■ Overview

CyberEDT is not just a website — it is the foundation of a cybersecurity ecosystem designed for:

- Students and beginners  
- Cybersecurity professionals  
- Businesses and organizations  

### The platform emphasizes:

- Practical learning  
- Real-world security tools  
- Awareness-driven defence
